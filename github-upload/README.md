## 💬 怎么用？

1. **打开链接** - 在手机或电脑浏览器中输入网址
2. **打字说话** - 在底部输入框输入你想说的话
3. **点发送** - 点击"发送"按钮或按回车
4. **等回复** - 稍等一下，我就会回复你啦！

**快捷按钮**：
- 👋 打个招呼
- 🌤️ 查天气
- 😄 讲个笑话
- ❓ 你能做什么
